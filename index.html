Text-Level Formatting

  • Use the em element to indicate stress emphasis.
  • Use the strong element to indicate importance, seriousness, or urgency.
  • Use the small element for side comments that could be small print.
  • Use the s element for text that is no longer accurate or relavant.
  • Use the i element for an alternate voice or mood.
  • Use the b element to draw attention to text for utilitarian purposes.
  • Use the mark element to highlight text for reference purposes.
  • Use the u element for unarticulated, though explicitly rendered, non-tetxtual annotation.
  • Use the abbr element for abbreviations such as dept., acronyms such as HTML, and initialisms such as ASCII.
  • Use the q element for inline quoted text.
  • Use the dfn element for words or phrases to be defined.
  • Use the cite element for the title of a work.
  • Use the data element for content that is machine-readable.
  • Use the time element for times such as October 31, 2025 that are machine-readable.
  • Use the code element for inline computer code.
  • Use the var element for variables such as E or x.
  • Use the samp element for computer output.
  • Use the kbd element for computer input.
  • Use the sub element for subscripts such as C6H12O6
  • Use the sup element for superscripts such as 1.602 × 10-19
  • Use the a element for internal hyperlinks and external hyperlinks.
